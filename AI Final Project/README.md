# Setup Instructions

1. Open a terminal in the project folder (`ML_project`)

2. Create a virtual environment with Python 3.10.11:
   - Windows: `python3.10 -m venv venv`
   - macOS/Linux: `python3.10 -m venv venv`

3. Activate it:
   - Windows: `venv\Scripts\activate`
   - macOS/Linux: `source venv/bin/activate`

4. Install required packages:
   ```bash
   pip install -r requirements.txt
   
5. ```bash
   python run.py
   
6. If you don't have Python 3.10.11, install it from https://python.org/
